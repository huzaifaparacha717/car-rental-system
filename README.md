# 🚗 Car Rental System

A modern **Car Rental Management System** built to simplify the process of renting vehicles online. This project provides an intuitive interface for customers to browse available cars, make reservations, and manage bookings while giving administrators complete control over vehicles, customers, and rental operations.

---

## 📌 Features

### 👤 User Features

* User Registration & Login
* Browse Available Cars
* Search & Filter Vehicles
* View Car Details
* Book a Car
* Rental History
* Profile Management

### 🛠️ Admin Features

* Secure Admin Dashboard
* Add, Edit & Delete Cars
* Manage Customers
* Manage Bookings
* Track Vehicle Availability
* Update Rental Status
* View System Records

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/huzaifaparacha717/car-rental-system.git
```

### 2. Open the Project

```bash
cd car-rental-system
```

### 3. Configure the Database

* Create a MySQL database.
* Import the provided SQL file (if available).
* Update your database credentials inside the configuration file.

### 4. Run the Project

If you're using **XAMPP**:

* Move the project to:

```
htdocs/
```

Start:

* Apache
* MySQL

Then visit:

```
http://localhost/car-rental-system
```
